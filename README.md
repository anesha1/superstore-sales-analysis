# Superstore Sales Analysis

End-to-end project: Kaggle sales dataset → Excel normalization → MySQL star schema → PowerBI dashboard.

**Tools:** Excel, MySQL Workbench, PowerBI

**Dashboard:** Open `Sales_visual.pbix` in PowerBI Desktop.

**Key Insights:**
- Total Sales: ~$118K
- Top Category: Technology (~59%)
- Top Segment: Consumer (~68%)
- Leading Region: West

**Files:**
- `Sales_visual.pbix`: PowerBI dashboard
- `Customer insights.png`, `Product analysis.png`, `Sales overview.png`: Dashboard screenshots
- `Superstore Sales Data Analysis Project Documentation.pdf`: Detailed project documentation
- `Separated_sheets.xlsx`: Normalized and sorted Excel sheet
- Raw data files: `Customers_superstore.csv`, `Order_Details_superstore.csv`, `Order_superstore.csv`, `Products_superstore.csv`, `Region_superstore.csv`

**Source:** [Kaggle Dataset](https://www.kaggle.com/datasets/poojacareer/sales-data-analysis-using-mysql-excel-and-power-bi)
